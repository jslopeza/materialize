Download

Materialize comes in two different forms. You can select which version you want depending on your preference and expertise. To start using materialize, all you have to is download one of the options below.

Materialize

This is the standard version that comes with both the minified and unminified CSS and JavaScript files. Use this if you are unfamiliar with Sass

Sass

This version contains the source SCSS files. By choosing this version you have more control over which components to include.
